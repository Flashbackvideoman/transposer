Song Transposer

On-line music transpose system
by Norman H. Strassner
Copyright (C) 2020, Norman H. Strassner

Users can load, drop or paste a text file where chords are placed directly above song text as in:

  As Tears Go By
  by The Rolling Stones
  
  [Intro] 
  G    A7    C    D7
  
  [Verse 1]
  G           A7             C    D7
    It is the evening of the da - ay.
  G           A7                 C     D7
    I sit and watch the children pla - ay.
  C               D7         G     D       Em Em7
    Smiling faces I can see,   but not for me.
  C                    Bm    C  D7
    I sit and watch as tears go by.

Note that text in brackets are optional and left untouched, as is the title lines.
The chords themselves will be bolded for display with HTML, but are plain text when downloaded or copyied.

You can edit the text like you would in a plain text editor. After changing key, and optionally editing the text, the Save button will save your changes and a "Download file" link will appear so that you can download the file. You can also copy and paste from the displayed song text

Use the +/- buttons, use the + or - on your keyboard to change song key

Present song in new window

Clicking on the "Present" button will open a new pop-up window (if pop-up windows are allowed) with the song in the current key.
Buttons labled "Larger" and "Smaller" will let you change the size of the text.

Uploading Song Text Files

File Selection

Click on the "Choose File" button and use the file dialog to select a music text file.
The file will be automatically uploaded and displayed.

Drop file

Drag and drop a music text file to the "Drop text file here" box.
The file will be automatically uploaded and displayed.

Paste file text

Copy all or part of the text from a text file (via some kind if text editor like Notepad, etc.)
and paste it in the text box on the right hand side.
